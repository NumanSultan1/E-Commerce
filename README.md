# Affordable Store - E-Commerce Website

Welcome to the **Affordable Store**! This is a simple, responsive e-commerce front-end application focused on providing premium tech products at affordable prices.

## 🚀 Features

* **Home Page**: A welcoming landing page featuring high-quality items, fast delivery propositions, and 24/7 support features.
* **Product Catalog**: A showcase of premium tech gadgets (e.g., mobile devices, CPUs, bags) with a clean grid layout.
* **Contact Area**: An easy-to-use contact form enabling users to reach out.
* **Responsive Design**: Designed to be responsive and visually appealing with a distinct, clean styling using custom CSS.

## 🛠️ Technology Stack

* **HTML5**: For crafting the semantic structure of the website.
* **CSS3**: Dedicated `style.css` file for professional, aesthetic styling, layouts, and typography.

## 📂 Project Structure

```text
E-Commerce/
├── index.html       # The main home page
├── product.html     # The products display page
├── contact.html     # The contact form page
├── style.css        # The core stylesheet for all pages
└── images/          # Product visualizations
    ├── bag.png
    ├── cpu.png
    └── mobile.png
```

## 💻 Getting Started

1. **Clone the repository** (if you haven't already):
   ```bash
   git clone <your-github-repo-url>
   ```
2. **Navigate to the project directory**:
   ```bash
   cd E-Commerce
   ```
3. **Run the Website**:
   Simply open `index.html` in your favorite web browser to see the store live!
